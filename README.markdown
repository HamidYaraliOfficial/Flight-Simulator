# 🇺🇸 English

## Flight Simulator

A modern desktop Flight Simulator built with Python and PyQt6. The project features a pseudo‑3D flight environment, multiple aircraft, weather systems, map selection, multilingual support, HUD instruments, flight physics, fuel management, and interactive controls.

## Features

- Pseudo‑3D cockpit flight view
- Multiple aircraft:
  - F‑16 Falcon
  - Boeing 747
  - Cessna 172
  - Concorde
- Multiple environments:
  - Clear Sky
  - Stormy
  - Mountain
  - Ocean
- Weather system:
  - Clear
  - Cloudy
  - Storm
  - Fog
- Real‑time flight physics
- Fuel consumption simulation
- Landing and crash detection
- Dynamic HUD and cockpit instruments
- FPS counter
- Dark and Light themes
- Multilingual interface:
  - English
  - Persian (Farsi)
  - Chinese

## Requirements

- Python 3.10+
- PyQt6

## Installation

```bash
pip install PyQt6
```

## Run

```bash
python "Flight Simulator.py"
```

## Controls

| Key | Action |
|------|---------|
| W / Up Arrow | Pitch Up |
| S / Down Arrow | Pitch Down |
| A / Left Arrow | Roll Left |
| D / Right Arrow | Roll Right |
| Q | Yaw Left |
| E | Yaw Right |
| Z | Increase Throttle |
| X | Decrease Throttle |
| ESC | Pause Menu |

## Project Structure

- Flight Physics Engine
- Flight Renderer
- HUD System
- Aircraft Selection
- Weather Management
- Theme Management
- Language System
- Main Menu & Settings

# 🇨🇳 中文

## 飞行模拟器

一个使用 Python 和 PyQt6 开发的现代桌面飞行模拟器。项目包含伪 3D 飞行环境、多种飞机、天气系统、地图选择、多语言界面、HUD 仪表以及飞行物理系统。

## 功能特点

- 伪 3D 驾驶舱视图
- 多种飞机
  - F‑16 Falcon
  - Boeing 747
  - Cessna 172
  - Concorde
- 多种地图场景
- 天气系统
  - 晴天
  - 多云
  - 风暴
  - 大雾
- 实时飞行物理
- 燃油消耗模拟
- 着陆与坠毁检测
- HUD 仪表系统
- FPS 显示
- 深色与浅色主题
- 多语言支持

## 系统要求

- Python 3.10+
- PyQt6

## 安装

```bash
pip install PyQt6
```

## 运行

```bash
python "Flight Simulator.py"
```

## 控制方式

| 按键 | 功能 |
|------|------|
| W | 抬头 |
| S | 低头 |
| A | 左滚转 |
| D | 右滚转 |
| Q | 左偏航 |
| E | 右偏航 |
| Z | 增加油门 |
| X | 减少油门 |
| ESC | 暂停菜单 |

## 项目组成

- 飞行物理引擎
- 渲染系统
- HUD 系统
- 飞机选择系统
- 天气系统
- 主题系统
- 多语言系统
